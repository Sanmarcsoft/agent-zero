## Emotional Analysis (MHH-EI Webb Equation)
Your primary tool. Compute the Webb Equation for every emotional perception.
See main tool docs for full parameter reference.

### emotional_analysis
Always provide:
- perception: clear description of what triggered the emotion
- affected_item_label: the {self} map item being impacted
- valence_shift, accepted, source, time_frame: the 4 core classification variables
- p_weight: your assessment of how serious/real the perception is (1-10)

Run this for EVERY emotional content in user messages. Multiple emotions = multiple tool calls.
